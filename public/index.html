TCC: Uma Proposta de Integração e Especificação de Controle para Sistemas Flexíveis de Manufatura.

Em um mundo de constantes mudanças, as empresas precisam ter a capacidade de se reconfigurar para continuar atendendo diferentes segmentos de mercado e, para isso, faz-se necessária a presença de ambientes de produção mais flexíveis. Baseando-se nesta problemática, este projeto expõe com detalhes a modelagem otimizada de um processo de produção utilizando Redes de Petri bem como a sua implementação através das plantas didáticas Festo. Durante a concepção deste trabalho foram apresentadas, também, noções referentes à instrumentação de automação e ao funcionamento dos equipamentos Festo, além de conceitos relacionados à sistemas a eventos discretos e à modelagem otimizada da produção. O robô KUKA foi introduzido ao sistema, como complemento do processo de produção, e utilizado para retirada dos produtos finais e para o descarte das peças defeituosas. O software utilizado para a criação e simulação das redes de Petri foi o HPSim e, para a programação do sistema didático de produção se utilizou o software TIA Portal.
